
## 窗口与应用切换

- Command + Tab：在已打开的 App 之间切换
- Command + `（数字 1 左边那个键）：在当前 App 的多个窗口间切换
- Command + H：隐藏当前 App 的所有窗口
- Option + Command + H：只保留当前 App，其他全部隐藏
- Command + M：最小化当前窗口到 Dock
- Option + Command + M：最小化该 App 的所有窗口
- Command + Option + Esc：打开「强制退出」窗口（卡死时用）

## Finder / 文件操作小技巧

- 空格：快速预览选中的文件（Quick Look）
- Command + Y：也是快速预览（在部分场景更好用）
- Option + Command + Y：把多个文件用幻灯片方式预览
- Command + Delete：把选中项目丢进废纸篓
- Shift + Command + Delete：清空废纸篓（配合 Option 跳过确认）
- Option + Command + V：在新位置「移动」文件（剪切效果）
    - 用法：先 Command + C 复制 → 到目标位置 → Option + Command + V

## 文本编辑「进阶」移动

这几个掌握了，写代码 / 写文档都很爽：

- Option + 左 / 右：按「单词」移动光标
    
- Option + Shift +左 / 右：按「单词」选择文字
    
- Command + 左 / 右：移动到行首 / 行尾
- Command + 上 / 下：移动到文档开头 / 结尾
- Control + A / E：移动到行首 / 行尾（很多终端 / Vim-like 软件好用）

## 搜索与启动

- Command + Space：打开 Spotlight（启动 App、算术、查单词都能用）
- Control + Command + Space：打开表情符号 / 符号面板
- Command + F：在当前页面 / 文档中搜索
- Command + L（浏览器）：光标直接跳到地址栏

## 截图与录屏（超实用）

- Shift + Command + 3：整个屏幕截图
    
- Shift + Command + 4：框选区域截图（按空格可切成窗口截图）
    
- Shift + Command + 5：打开截图与录屏工具（可以选区域录屏）


## 几个组合使用的实战示例

- 整理桌面：  
    Command + Option + H（只留当前 App）→ Command + H（也隐藏）→ 看到桌面
- 阅读 PDF / 网页：  
    用 Option + 左 / 右 快速跳单词，Command + 上 / 下 快速跳到头或尾
- 频繁切换 IDE / 浏览器：  
    Command + Tab 在 App 间切，Command + ` 在同 App 的窗口间切
